PlankSetting
============

Application to customize Plank dock

Dependence
==========

unrar

unzip

python3

libgtk-3-0


Install
=======

open terminal and type:
chmod +x install.sh    #to make the installer executable

./install.sh

or

sh install.sh

Uninstall
=========

open terminal and type:

chmod +x uninstall.sh    #to make the uninstaller executable

./uninstall.sh

or

sh uninstall.sh
