PlankSetting Pages.

This is a simple application that allow users to customize plank dock easily, Features:

  • Customizing the behavior of the dock.
  • Changing the dock theme.
  • Adding new themes.
  • Removing the undesirable themes.

Note: You need Plank to be installed first.
